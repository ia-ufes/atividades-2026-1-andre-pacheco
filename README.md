# Repositório exemplo para entrega de atividades

Este é um exemplo de repositório para entrega de atividades da disciplina.
Observe que ele está público para visualização, mas o seu repositório deve ser **PRIVADO**!

Cada pasta deve conter a sua atividade em questão. Neste exemplo existem 3 pastas:
- `A1-A-TOPSIS`
- `A2-NOME`
- `A3-NOME`

Os nomes das atividades serão definidos no momento em que elas são divulgadas. Neste caso, a 1ª atividade é a do A-TOPSIS.

Siga a risca esse padrão para que suas atividades sejam corrigidas corretamente.
